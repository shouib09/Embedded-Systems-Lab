# Embedded Systems Lab

This repository contains Embedded Systems laboratory experiments performed using:

- AT89C51
- Arduino Uno
- LPC2148
- Proteus
- Keil µVision
- Embedded C
- SystemC
- RTOS

---

## Software Used

- Keil µVision
- Proteus 8
- Arduino IDE
- Flash Magic
- LPC2148 Development Kit

---

## Experiments

### AT89C51

- LED Blinking
- Square Wave Generation
- Stepper Motor
- Seven Segment Display

### Arduino

- Manual Counter
- LCD Interface
- Serial Communication
- MQ2 Gas Sensor
- Ultrasonic Sensor
- PIR Sensor

### Embedded C

- GPIO
- 16-bit Multiplication
- Array Addition
- UART
- PWM

### ARM LPC2148

- LED Blinking
- Switch Interface
- Seven Segment
- UART
- ADC

### SystemC

- SC_THREAD
- Producer Consumer
- Transaction Level Modeling
- Binary Semaphore
- Counting Semaphore

### RTOS

- Interrupt Handling
- Task Scheduling
